
          LibOSM

 LibOSM is a library for downloading, parsing,
 storing, searching and manipulating
 OpenStreetMap-xml-data.
 It also features a plugin- and configuration-system
 to make it easy to build applications that work
 with OSM-data.

 http://travelingsales.sourceforge.net

 ------------ building ---------------

 with ANT: "ant" or "../traveling_salesman/build/apache-ant-1.7.0/bin/ant"
             if it is not in your PATH.
   * the output will be in ./dist/

 with maven: "mvn install" or "./maven/bin/mvn install"
             if it is not in your PATH.
   * the output will be in ./target/
